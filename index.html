02 — Core Research Items

4つの研究開発項目と
技術的アプローチ

01 Foundation

3DGS環境表現の構築と評価

レンダリング品質だけでなく、ロボットの「再局在化（Relocalization）精度」や「経路計画成功率」を主眼に置いたパラメータ最適化を実施。

主要KPI: Latency / GPU memory footprint / Incremental update cost

02 Intelligence

Semantic Integration

Neuro-Symbolic Approach:
LLMは「意味解釈」に専念させ、幾何計算は既存アルゴリズムで担保。音声アノテーションで現場負担を軽減。

03 Interoperability

BIM 相互運用

完全自動化ではなくHuman-in-the-loopによる支援技術。3DGSとBIMを重畳し、差分を可視化・管理する。

04 Governance

Trusted Digital Twin 基盤

既存の「秘密計算」共同研究を発展させ、空間データのガバナンス（真正性・履歴・権利）を確立する。
